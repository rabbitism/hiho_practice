# 1788 : 偶数长度回文子串

Time Limit:5000ms
Case Time Limit:1000ms
Memory Limit:256MB

## Description

给定一个小写字母字符串，请判断它是否有长度为偶数的非空连续回文子串

## Input

输入包含多组数据。

每组数据包含一行一个小写字母字符串 S

1 ≤ |S| ≤ 105

## Output

对于每组数据如果存在，输出YES，否则输出NO

### Sample Input

```bash
cabbad
ababa
```

### Sample Output

```bash
YES
NO
```
