# 1824 : 右转九十度

Time Limit:10000ms
Case Time Limit:1000ms
Memory Limit:256MB

## Description

假设一个机器人在笛卡尔坐标系上。它从(X1, Y1)移动到了(X2, Y2)，然后向右转90度，继续前进。  

请你计算这个机器人继续前进过程中最先经过的整点是哪一个？

## Input

四个整数X1, Y1, X2, Y2。

-1000000 ≤ X1, Y1, X2, Y2 ≤ 1000000 保证(X1, Y1)和(X2, Y2)是不同的点。

## Output

两个整数X和Y代表最先经过的整点坐标。

### Sample Input

```bash
0 0 1 2
```

### Sample Output

```bash
3 1
```
