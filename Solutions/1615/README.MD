# 1615 : 矩阵游戏II

Time Limit:10000ms
Case Time Limit:1000ms
Memory Limit:256MB

## Description

给定一个NxN的整数矩阵，小Hi每次操作可以选择两列，将这两列中的所有数变成它的相反数。

小Hi可以进行任意次操作，他的目标是使矩阵中所有数的和尽量大。你能求出最大可能的和吗？

## Input

第一行一个整数N。  

以下N行，每行N个整数Aij。  

对于30%的数据，2 ≤ N ≤ 10  

对于100%的数据，2 ≤ N ≤ 200， -1000 ≤ Aij ≤ 1000

## Output

最大可能的和

### Sample Input

```txt
4
-1 1 1 2 
-2 -3 1 2  
-3 -2 1 2  
-4 -1 1 2
```

### Sample Output

```txt
27
```
