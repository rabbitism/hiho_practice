# 1642 : 三角形面积和

Time Limit:10000ms

Case Time Limit:1000ms

Memory Limit:256MB

## Description

如下图所示，在X轴上方一共有N个等腰直角三角形。这些三角形的斜边与X轴重合，斜边的对顶点坐标是(Xi, Yi)。

```bash

             (11,5)
      (4,4)    /\
        /\(7,3)  \
       /  \/\/    \
      /   /\/\     \
     /   / /\ \     \
------------------------->
```

你能求出这些三角形覆盖的面积之和吗？ (重叠部分只算一次)

## Input

第一行包含一个整数N。  

以下N行每行包含两个整数(Xi, Yi)，代表第i个三角形顶点的坐标。

对于30%的数据，1 ≤ N ≤ 100, 0 ≤ Xi, Yi ≤ 100  

对于100%的数据，1 ≤ N ≤ 100000，0 ≤ Xi, Yi ≤ 100000

## Output

覆盖的面积，保留2位小数。

### Sample Input

```bash
3
7 3
4 4  
11 5
```

### Sample Output

```bash
42.00
```
