# 1842 : AB字符串

Time Limit:10000ms

Case Time Limit:1000ms

Memory Limit:256MB

## Description

给定两个整数N和M，请你构造一个恰好包含N个'A'和M和'B'的字符串，并且满足字符串中不包含子串"AAA"和"BBB"。  

如果有多个字符串满足条件，你可以输出其中任意一个；如果没有满足条件的字符串，输出-1。

## Input

两个整数N和M。  

0 <= N, M <= 1000

## Output

如果有多个字符串满足条件，输出其中任意一个；如果没有满足条件的字符串，输出-1。

### Sample Input

```bash
5 2
```

### Sample Output

```bash
ABAABAA
```
